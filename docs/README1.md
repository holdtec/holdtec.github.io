## 📦 文档部署

```bash
#克隆文档
git clone https://github.com/holdtec/holdtec.github.io.git
#安装依赖 注意使用node 6版本 高版本不支持
cd genesis-accounting-user-manual
nvm use 6
# 项目已包含node_modules 且已定制
#不要install！ 不要install！不要install！

#gitbook install
#本地运行
```
