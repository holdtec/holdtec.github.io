安装与注册

如何下载和安装

万能复式记账小程序无需下载和安装，只需打开微信，搜索“万能复式记账”小程序，即可立即使用。用户可以随时随地通过微信访问小程序，享受便捷的记账服务。

或者通过微信扫一扫，扫描小程序码直接进入小程序。

用户注册和登录

用户注册和登录过程简单快捷，无需授权，不获取任何敏感信息。用户首次进入小程序时，系统会自动获取微信官方返回的一串无任何意义的字符串作为用户标识。此标识仅用于区分不同用户，不涉及用户的个人隐私信息。
