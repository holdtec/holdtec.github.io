  ### 📂 项目简介

  ![万能复式记账](https://cdn.jsdelivr.net/gh/tgineer/images/images/gh_6b711893fde2_344-20240710215706657-20241021113209444.jpg)

  - 项目名称：万能复式记账微信小程序
  - 简介：一个基于 Beancount 记账理念，支持多账本、多成员、会员管理和进销存功能的复式记账系统。
  - 主要功能：记账管理、数据分析、会员管理、门店促销活动管理、进销存管理等。
